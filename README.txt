Программа получает на вход папку и слово, которое необходимо найти в файлах этой папки, включая вложенные, с выбранным расширением.
Результаты поиска отображаются в левой части окна в виде дерева файловой системы, где только те файлы, в которых был обнаружен заданный текст.
В правой части окна для каждого найденного файла создается вкладка с возможностью навигации по тексту.
